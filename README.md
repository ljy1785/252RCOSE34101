# OS Assignment Git Practice

Simple C programs used to practice Git commits in WSL2.

- git_assignment.c: first practice file
- practice/hello.c: hello world
- practice/add.c: addition
- practice/loop.c: for-loop example
- practice/array_sum.c: array sum
- practice/max.c: max value
- practice/strlen_manual.c: manual strlen
- practice/struct_student.c: struct example
- practice/pointer_basic.c: pointer basics
- practice/factorial.c: factorial function
- practice/swap.c: swap two integers using pointers
